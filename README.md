# AI-Driven Customer Analytics Platform
## Overview

AI-Driven Customer Analytics Platform uses Machine Learning,
Statistical Analysis, and Data Visualization to analyze customer behavior,
predict purchases, identify churn risk, and generate recommendations.

## Features

- Customer Segmentation
- Purchase Prediction
- Churn Prediction
- Recommendation Engine
- Interactive Dashboard
- Statistical Analysis
- Graph Visualization

## Technology Stack

- Python
- Streamlit
- Pandas
- NumPy
- Scikit-Learn
- Plotly
- Joblib

## Workflow
